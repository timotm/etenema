Etenemä v0.1 - ladataan, odota...

Mikä?

Sykemittarin ja gps:n kanssa tehdyille, Stravaan ladatuille juoksusuorituksille voidaan tällä palvelulla visualisoida syke-etenemä-indeksiä.

Indeksin arvoja piirretään väliaikakohtaisesti (lap) seuraavalla kaavalla: matka (m) / ( aika (min) * ( keskisyke - leposyke ) ) Indeksigraafin aikasarjasta voi nähdä kunnon kehittymisen ajan funktiona. Juoksuvauhti / indeksi -graafista puolestaan voi tehdä päätelmiä eri juoksunopeuksien tehokkuudesta.
